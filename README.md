# Finance Dashboard UI

## Overview
A simple finance dashboard built with **HTML, CSS, and JavaScript**.  
It allows users to view financial summaries, explore transactions, and gain insights into spending patterns.  
The project demonstrates frontend skills including UI design, state management, and role-based behavior.

## Features
- **Summary Cards**: Total Balance, Income, Expenses
- **Charts**:  
  - Line chart for balance trend  
  - Pie chart for spending breakdown  
- **Transactions Table**:  
  - Displays Date, Amount, Category, Type  
  - Search and filter by category  
  - Sorting options  
- **Role-Based UI**:  
  - Viewer → read-only access  
  - Admin → can add transactions via form  
- **Insights Section**:  
  - Highest spending category  
  - Monthly comparison  
  - Observations from data

## Tech Stack
- HTML, CSS, JavaScript
- Chart.js (for visualizations)
